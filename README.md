# 新生儿护理记录（移动端 Web）

一个基于 Java Spring Boot 的轻量项目，用于记录：

- 大便（仅时间，快速打卡）
- 小便（仅时间，快速打卡）
- 奶粉（开始奶粉 / 结束奶粉，结束时填写奶量且必须是 30 的倍数）
- 母乳（开始母乳 / 结束母乳，自动计算时长；可选填写毫升）
- 体重（每日可多次记录，单位克）

并支持：

- 按日期查看每次打卡详情（包含开始/结束时间）
- 按天统计：
  - 奶粉：总毫升
  - 母乳：总时长 + 总毫升（若有填写）
  - 大小便：次数
  - 体重：记录次数与当日最新体重

> 数据会持久化到 `data/events.json`，服务停止后重启不会丢失打卡记录，包括进行中的奶粉/母乳状态。

## 一键预览

```bash
./preview.sh
```

默认访问：`http://localhost:8080`

可选参数：

- `--port=8090` 自定义端口
- `--dry-run` 只打印启动命令，不实际启动

## 启动

```bash
mvn spring-boot:run
```

打开浏览器访问：`http://localhost:8080`
